(Jongin Park) With "use strict":

Using a variable/object without declaring it, is not allowed.

Deleting a variable (or object, function) is not allowed.

Duplicating a parameter name is not allowed.

Octal numeric literals are not allowed.

Octal escape characters are not allowed.

Writing to a read/get-only property is not allowed.

Deleting an udeletable property is not allowed.

The string "eval" cannot be used as a variable.

The string "arguments" cannot be used as a variable.

The with statement is not allowed.

For security reasons, eval() is not allowed to create variables in the scope from which it was called.

Activate debugging in your browser (F12) to see the error report.
